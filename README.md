# Личен Уебсайт - Cybersecurity Tools & Resources Hub

## Описание

Уебсайт за онлайн инструменти за сигурност, предназначен за разработчици и специалисти по киберсигурност. Платформата предоставя полезни инструменти и ресурси за подобряване на сигурността на приложенията и данните.

## Функционалности

### 1. Инструменти за сигурност
- **Генератор на пароли** - Генерира сигурни, случайни пароли
- **Проверка на силата на паролите** - Анализира силата на паролите според дължината и сложността

### 2. Форум
- Публичен форум за дискусии
- Потребителите могат да задават въпроси, споделят идеи и обсъждат проблеми по киберсигурност
- CRUD операции (създаване, четене, редактиране, изтриване на коментари)
- Сортиране по дата и потребителско име

### 3. Изтегляния
- Ресурси за сигурност
- Ръководства и чеклисти
- Скриптове и инструменти

### 4. За мен
- Автобиография
- Образование и опит
- Технически умения

## Технологии

- **Backend:** ASP.NET Core MVC (.NET 10.0)
- **Frontend:** HTML5, CSS3, JavaScript, jQuery
- **Database:** MySQL (Pomelo.EntityFrameworkCore.MySql)
- **Framework:** Bootstrap 5 (Responsive Design)
- **Ajax:** Асинхронни операции за CRUD функционалности

## Изисквания

- .NET 10.0 SDK
- MySQL Server
- Visual Studio 2022 или Visual Studio Code
